# Docsify-Google-Adsense
## Usage
### Load JS
```HTML
<script src="//cdn.jsdelivr.net/gh/kb1000fx/ToolBox/docsify-google-adsense/docsify-google-adsense.js"></script>
```
### Config
```HTML
<script>
    window.$docsify = {
        ...
        adClient: YOUR-PUB-ID,
        adHeaderSlot: YOUR-SLOT-ID,
        adFooterSlot: YOUR-SLOT-ID,
        ...
    }
</script>
```
